Machine Learning Analysis

In looking at the data, we chose to use three Machine Learning models to try to predict Gold prices through 2030. The three algorithms we chose were:

  • Linear Regression
  • Random Forest Regression
  • Long Short-Term Memory Neural Network (LSTM)

The Linear Regression model proved to be not viable for the data as there were too many variables for the regression model to accurately predict any future data. As seen below, the Random Forest Regression model and LSTM Model were able to produce better results.

Our final review showed that the LTSM Model was the preferred and most accurate model out of the three we chose to explore.

Final Projected Data

Our final projected data shows Gold prices through 2030 using the LTSM Model, which had the highest correlation between Test and Predicted data when training the model. The LTSM Model projects Gold prices to drop between 2022 and 2024 to ~$1275 - $1300. We then see the price trajectory from 2024 - 2028 steadily rising to ~$1525 before descending to ~$1475 in 2030.

In looking at historical price data, these values are on trend / within realistic values for Gold.
